== Description ==

The [Linux Promotional Plugin](http://www.itsqueeze.com/2011/06/wordpress-linux-promotional-plugin/ "Linux Promotional Plugin") is a plugin that shows a informative message alerting Windows and Mac users about the advantages of Linux, and providing links to choosen distros.

You can choose from 2 types of promotions:

* TOP, a discreet top bar
* FULL, a fullscreen promotional message

See the new [Linux Promotional Plugin](http://www.itsqueeze.com "Linux Promotional Plugin") for further information.

All suggestions for improvements are welcome!

== Installation ==

1. Upload the full directory into your /wp-content/plugins/ directory, or install it through the admin interface
2. Choose the type of promotion in the options page and save
3. DONE!
